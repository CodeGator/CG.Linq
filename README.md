#### What does it do?
This package contains LINQ extensions used by other CodeGator packages.

#### Commonly used types:
* CG.Linq.Criteria
* CG.Linq.ExpressionExtensions
* CG.Linq.PredicateBuilder
* CG.Collections.Generic.EnumerableExtensions

#### What platform(s) does it support?
* .NET 6.x or higher.
